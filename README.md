# Concord Website

This is the source code of the website for the workflow and integration server
Concord.

The site is available at
[https://concord.walmartlabs.com](https://concord.walmartlabs.com) (not yet but
very soon).

## Build

```
gem install bundler
bundle install
./run.sh
```
